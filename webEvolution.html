What is HTML?

If you've ever built a website, you're probably familiar with HTML. But what exactly is HTML? Why use HTML to create a website?

Definition

HTML stands for Hyper Text Markup Language, which is the standard markup language used to display documents in a web browser like Google Chrome or Edge. HTML is typically used in conjunction with other technologies such as Cascading Style Sheets (CSS) and JavaScript.

The structure within HTML consists of a series of elements, which function to provide information to the web browser about how to display content. An HTML document contains many types of elements such as paragraphs, inputs, headings, and so on, including an opening tag, content, and a closing tag, with some elements not requiring a closing tag.

Brief History

HTML was first discussed in 1980 by Tim Berners-Lee while working at CERN (European Organization for Nuclear Research). At that time, he proposed creating a system named ENQUIRE to manage company documents.

Later in 1990, Berners-Lee, along with his team at the same company, began looking for funding for the project which would eventually lead to the creation of HTML. HTML was first published in 1991 with only 18 tags, but today it has evolved significantly.

Conclusion

HTML has come a long way since Tim Berners-Lee first proposed it in 1980 while working at CERN. From managing documents in the ENQUIRE system to becoming the backbone of web development, HTML has undergone substantial evolution.
